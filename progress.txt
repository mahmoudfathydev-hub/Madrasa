MADRASA PROJECT REPORT (progress.txt)

1) PROJECT TYPE / STACK
-----------------------------------------------------------------------
- App type: Single-page application (SPA)
- Framework: React (React 19)
- Bundler/dev server: Vite
- Styling: Sass/SCSS
- Routing: react-router-dom
- Internationalization (i18n): i18next + react-i18next + i18next-browser-languagedetector
- Animations:
  - Framer Motion (JS-driven component animations)
  - CSS/SCSS keyframe animations

3) SOURCE FOLDER CONFIGURATION (src/)
-----------------------------------------------------------------------
Top-level src structure:
- App.jsx
  - Defines routes (react-router-dom Routes/Route)
  - Currently includes:
    - /, /login -> Login page
    - /signup -> Signup page
    - /home -> Home page
    - /about-us -> AboutUs page
  - Wraps all routes with Navbar and Footer
  - Sets document direction (dir) based on language (rtl for ar)
- main.jsx
  - React entrypoint
  - Uses BrowserRouter
  - Imports global styles: index.scss
  - Initializes i18n via import "./i18n"
- index.scss
  - Global CSS variables, resets, base styles, container width
  - Imports Google Fonts
- i18n.js
  - i18next config
  - Loads locale JSON files from src/locales
  - Uses LanguageDetector + initReactI18next
- components/
  - Reusable UI sections/components (see section 4)
- pages/
  - Page-level routes (see section 5)
- locales/
  - Translation JSON:
    - locales/en/En.json
    - locales/ar/Ar.json
- images/
  - Image assets used by sections/components
- data/
  - Placeholder JS modules (currently empty files):
    - coursesData.js
    - teachersData.js
    - quizData.js
    - localStorageHelpers.js

How to use/install (src):
- Nothing to separately install per folder.
- All folders are part of the same Vite + React app.
- Add/modify components/pages by importing them into routes or other components.

7.B) FRAMER MOTION ANIMATIONS (used in code)
- src/components/Testimonials/Testimonials.jsx
  - Uses AnimatePresence + motion.div
  - slideVariants:
    - enter(direction): x +/- 1000, opacity 0, scale 0.8, rotateY +/- 45
    - center: x 0, opacity 1, scale 1, rotateY 0
    - exit(direction): x +/- 1000, opacity 0, scale 0.8, rotateY +/- 45
  - Transition:
    - x: spring (stiffness 300, damping 30)
    - opacity: duration 0.001
  - Drag/swipe interaction: drag="x" + onDragEnd to paginate

- src/components/TeacherCard/TeacherCard.jsx
  - heading: initial {opacity 0, y -20} -> whileInView {opacity 1, y 0}, duration 0.8
  - containerVariants:
    - hidden {opacity 0}
    - visible {opacity 1, transition {staggerChildren 0.2}}
  - itemVariants:
    - hidden {opacity 0, y 30}
    - visible {opacity 1, y 0, transition {duration 0.6, ease 'easeOut'}}
  - Hover: whileHover {y -10, scale 1.02, transition {duration 0.3}}

- src/components/Aspirations/Aspirations.jsx
  - Same pattern as TeacherCard:
    - heading initial/whileInView
    - containerVariants (staggerChildren)
    - itemVariants (fade+slide up)
    - hover lift/scale

- src/components/ComponentsAboutUS/AboutFeatures/AboutFeatures.jsx
  - Same pattern:
